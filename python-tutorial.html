1   Lab Introduction

There are at least three occasions we will need Python in this course: (1) Python will be used to write programs to run on the desktop to interact or control the mbed board. (2) Python is used in Tensorflow training framework. (3) Also MicroPython (a version of Python for embedded system) is used in OpenMV H7+ for embedded vision.
