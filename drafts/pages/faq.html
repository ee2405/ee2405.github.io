2   Git FAQ

Homework creation/submission flow

In this course, reports and codes are submit to Gitlab. Please use the following steps to submit:

  1. First please create a Gitlab project.
  2. Before any submission (check in), an empty Gitlab project (just created above) may require you to create a readme file.
  3. For each homework or project, you may New a Directory and upload your works there.

How to add/change files in a Gitlab project

There are two ways.

  1. via web interface
  2. via Git commandline (you will need to use a terminal in Linux/Mac or install a terminal application in Windows).
